# Roadmap

## Current focus

- Stabilize the open-source release.
- Improve documentation and local setup.
- Expand automated tests for auth, payments, delivery, and admin flows.
- Harden security around webhooks, sessions, 2FA, and secrets.

## Planned

- Demo deployment guide.
- Docker-based local development setup improvements.
- More provider adapters for payments and delivery.
- Better multi-brand configuration examples.
- Admin UX improvements.
- Contribution-friendly issue labels and starter tasks.

## Not planned yet

- Marketplace-style multi-seller accounting.
- Enterprise SaaS billing.
- Public cloud hosting service.
